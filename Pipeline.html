Class: Pipeline

Pipeline

Pipelines can be used to process elements (data/objects/functions) through a chain of multiple processing steps, called stages.

Each stage is a function that processes a single element at a time. A stage may either modify the given input or construct and return new data to be passed through the rest of the pipeline.

Example

Create a pipeline that calculates f(x) = 2x + 15

var polynom = new Pipeline()
        .stage(function(x){return 2x})
        .stage(function(x){return x + 15});
polynom.consume(3)
